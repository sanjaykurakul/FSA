# Full Stack Application Development - Tasks 1 to 24 (Java-free version)

This package contains **Task 1 to Task 24** converted to a **Java-free stack** as requested.
Instead of Java/Spring, the backend-oriented tasks use:
- HTML5, CSS3, JavaScript
- Node.js + Express
- SQLite / SQL
- JSON / REST APIs
- Docker / YAML / Git / CI-CD configs where needed

Each task folder contains:
- `README.md` : aim, files needed, run steps
- required code files
- `output_task_X.png` : a sample output image/mock output

> Note:
> Tasks 8 to 18 were originally Spring / Java oriented in the syllabus.
> They have been re-implemented here using **Node.js / Express / Sequelize/SQLite style alternatives**
> so that the concepts remain covered without using Java.
